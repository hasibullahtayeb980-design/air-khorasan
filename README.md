# Getting Started
## Using Docker

### Prerequisites
- [Download and Install Docker](https://www.docker.com/products/docker-desktop/)

Then simply execute the following commands in the root directory of this project (where `compose.yml` is located):
```
docker compose up
```
It will setup all dependencies for you in Docker containers.

## Without Docker
If you experience any issues with Docker then you can [setup the project without Docker here](docs/without-docker.md).
